Restaurant Availability Engine es una solución backend de nivel empresarial construida con Django y Django REST Framework.

Diseñado para gestionar la disponibilidad y reservaciones en tiempo real, implementa una arquitectura modular basada en motores de reglas (Facade Pattern). Permite manejar horarios dinámicos, temporadas, excepciones de calendario y control de capacidad granular, ofreciendo una API robusta y escalable lista para integración con clientes frontend.

Características clave: Arquitectura limpia, Docker-ready, i18n, y cobertura de tests exhaustiva.
